Teste de Iframes com Data URI

O que é Data URI em iframes?

Assim como nas imagens, Data URI permite incorporar o conteúdo diretamente no atributo src do iframe, em vez de referenciar um arquivo externo. Para iframes, isso geralmente significa incorporar HTML diretamente.

Vantagens e Desvantagens de Data URI em iframes

Vantagens

  • Reduz o número de requisições HTTP
  • Conteúdo disponível imediatamente sem requisições adicionais
  • Útil para pequenos trechos de HTML
  • Funciona offline

Desvantagens

  • Aumenta o tamanho do HTML da página principal
  • Difícil de manter e atualizar
  • Limitações de tamanho em alguns navegadores
  • Problemas de segurança potenciais

Sobre este teste

Esta página testa como o crawler lida com iframes que usam Data URI em vez de URLs convencionais.

O que o crawler deve fazer:

  • Detectar iframes que usam Data URI
  • Possivelmente extrair e analisar o conteúdo HTML incorporado
  • Identificar o tipo de conteúdo (HTML, texto, etc.)
  • Verificar se os iframes têm atributo title para acessibilidade
