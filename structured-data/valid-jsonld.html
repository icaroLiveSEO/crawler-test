Sobre este teste

Esta página contém dados estruturados JSON-LD válidos que seguem o schema.org. O objetivo é testar como o crawler extrai e interpreta esses dados.

O que o crawler deve fazer:

  • Detectar o script JSON-LD
  • Extrair os dados estruturados
  • Identificar o tipo de entidade (Article)
  • Validar os dados contra o schema.org
