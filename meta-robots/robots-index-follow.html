Teste de Meta Robots - Index, Follow

Diretiva robots presente nesta página:

<meta name="robots" content="index, follow" />

Sobre este teste

Esta página contém a diretiva meta robots configurada para permitir indexação e seguimento de links.

O que o crawler deve fazer:

  • Indexar esta página (permitir que apareça nos resultados de busca)
  • Seguir os links encontrados nesta página

Esta é a configuração padrão para a maioria dos sites. Mesmo sem esta meta tag, os motores de busca geralmente assumem "index, follow" como comportamento padrão.
