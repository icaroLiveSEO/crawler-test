Teste de X-Robots-Tag via HTTP Header

Esta página usa X-Robots-Tag via HTTP Header

O servidor está configurado para enviar o seguinte cabeçalho HTTP com esta página:

X-Robots-Tag: noindex
Nota: Para que este teste funcione corretamente, o servidor web deve estar configurado para enviar o cabeçalho HTTP X-Robots-Tag com esta página. Isso geralmente requer configuração no servidor web (Apache, Nginx, etc.) ou em um arquivo .htaccess.

Sobre este teste

Esta página demonstra o uso do cabeçalho HTTP X-Robots-Tag em vez da meta tag robots. Ambos têm o mesmo efeito, mas o cabeçalho HTTP pode ser aplicado a qualquer tipo de arquivo, não apenas a páginas HTML.

O que o crawler deve fazer:

  • Detectar o cabeçalho HTTP X-Robots-Tag
  • NÃO indexar esta página (devido à diretiva noindex)

Vantagens do X-Robots-Tag:

  • Pode ser aplicado a qualquer tipo de arquivo (PDF, imagens, vídeos, etc.)
  • Pode ser configurado em nível de servidor para grupos de arquivos
  • Útil para arquivos não-HTML que precisam de controle de indexação

Configuração de exemplo para Apache (.htaccess):

<Files "x-robots-tag.html">
    Header set X-Robots-Tag "noindex"
</Files>
