Guia de Tags Semânticas HTML5

O HTML5 introduziu várias tags semânticas que ajudam a definir a estrutura e o significado do conteúdo da web. Usar essas tags corretamente melhora a acessibilidade, o SEO e a manutenção do código.

Tag <header>

A tag <header> representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação. Geralmente contém o logotipo, título do site e navegação principal.

Uso correto: Para cabeçalhos de página ou seções importantes.
<header>
  <h1>Título do Site</h1>
  <p>Slogan ou descrição</p>
</header>

Tag <nav>

A tag <nav> define uma seção de navegação. É usada para menus de navegação principais, menus laterais, índices de página e outras listas de links de navegação.

Uso correto: Para conjuntos de links de navegação principais.
<nav>
  <ul>
    <li><a href="#">Início</a></li>
    <li><a href="#">Sobre</a></li>
    <li><a href="#">Contato</a></li>
  </ul>
</nav>

Tag <main>

A tag <main> especifica o conteúdo principal de um documento. O conteúdo dentro do elemento <main> deve ser único ao documento e não deve incluir conteúdo repetido em outros documentos, como barras laterais, links de navegação, informações de copyright, logotipos e formulários de pesquisa.

Uso correto: Deve haver apenas um elemento <main> por página.
<main>
  <article>...</article>
  <aside>...</aside>
</main>

Tag <article>

A tag <article> especifica conteúdo independente e autocontido. Um artigo deve fazer sentido por si só e deve ser possível distribuí-lo independentemente do resto do site.

Uso correto: Para posts de blog, notícias, comentários de usuários, etc.
<article>
  <h2>Título do Artigo</h2>
  <p>Conteúdo do artigo...</p>
</article>

Tag <section>

A tag <section> define uma seção em um documento. De acordo com a documentação HTML5, uma seção é um agrupamento temático de conteúdo, tipicamente com um cabeçalho.

Uso correto: Para dividir conteúdo em seções temáticas.
<section>
  <h2>Título da Seção</h2>
  <p>Conteúdo da seção...</p>
</section>

Tag <aside>

A tag <aside> define um conteúdo levemente relacionado ao conteúdo ao redor. O conteúdo aside é frequentemente colocado como uma barra lateral.

Uso correto: Para conteúdo relacionado, barras laterais, anúncios, etc.
<aside>
  <h3>Artigos Relacionados</h3>
  <ul>
    <li><a href="#">Artigo 1</a></li>
    <li><a href="#">Artigo 2</a></li>
  </ul>
</aside>

Tag <footer>

A tag <footer> define um rodapé para um documento ou seção. Um elemento footer geralmente contém informações sobre o autor, direitos autorais, links para documentos relacionados, etc.

Uso correto: Para rodapés de página ou seções.
<footer>
  <p>© 2023 Meu Site. Todos os direitos reservados.</p>
  <address>Contato: exemplo@email.com</address>
</footer>

Benefícios do HTML Semântico

  • Melhor acessibilidade para leitores de tela
  • Código mais fácil de manter
  • Melhor SEO e indexação por motores de busca
  • Facilita o estilo com CSS
  • Torna o código mais legível
