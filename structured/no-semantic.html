Bem-vindo ao nosso site
Este é um site de exemplo que não utiliza tags semânticas HTML5. Isso torna difícil para os motores de busca entenderem a estrutura do conteúdo.
Subseção 1
Este conteúdo deveria estar em uma tag <section> ou <article>. Sem essas tags, os crawlers têm dificuldade em identificar a importância relativa do conteúdo.
Subseção 2
Outro problema é a falta de hierarquia clara nos títulos. Não estamos usando h1, h2, h3, etc. de forma adequada.
