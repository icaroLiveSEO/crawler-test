Introdução às Tags de Texto HTML

O HTML oferece uma variedade de tags para formatar e estruturar texto. Usar essas tags corretamente não apenas melhora a aparência do seu conteúdo, mas também ajuda na acessibilidade, SEO e na compreensão geral do seu texto.

Nesta página, demonstraremos o uso de várias tags de texto HTML, incluindo cabeçalhos (h1-h6), parágrafos, listas, citações, ênfase e muito mais.

Hierarquia de Cabeçalhos (h1-h6)

Os cabeçalhos HTML são usados para definir a estrutura e hierarquia do conteúdo. Eles vão de h1 (o mais importante) até h6 (o menos importante). Uma página bem estruturada deve ter apenas um h1 e seguir uma hierarquia lógica.

Este é um cabeçalho h1

O h1 deve ser usado apenas uma vez por página, geralmente para o título principal.

Este é um cabeçalho h2

O h2 é usado para seções principais dentro do conteúdo.

Este é um cabeçalho h3

O h3 é usado para subseções dentro de uma seção h2.

Este é um cabeçalho h4

O h4 é usado para divisões menores dentro de uma subseção h3.
Este é um cabeçalho h5
O h5 é raramente usado, apenas para estruturas muito detalhadas.
Este é um cabeçalho h6
O h6 é o nível mais baixo de cabeçalho, usado apenas em casos especiais.

Parágrafos e Quebras de Linha

A tag <p> é usada para definir parágrafos, que são blocos de texto separados por espaços em branco. Para quebras de linha dentro de um parágrafo, usamos a tag <br>.

Ênfase e Destaque

Para dar ênfase ao texto, usamos as tags <em> e <strong>. A tag <em> indica ênfase (geralmente renderizada em itálico), enquanto <strong> indica forte importância (geralmente renderizada em negrito).

Este texto contém ênfase e forte importância.

Você também pode combinar as tags: Este texto é enfatizado e importante.

Listas Ordenadas e Não Ordenadas

As listas são usadas para agrupar itens relacionados. Existem dois tipos principais: listas ordenadas (<ol>) e listas não ordenadas (<ul>).

Lista Ordenada

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
    1. Subitem 3.1
    2. Subitem 3.2
  4. Quarto item

Lista Não Ordenada

  • Item de lista
  • Outro item
  • Item com sublista
    • Subitem
    • Outro subitem
  • Último item
