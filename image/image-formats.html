Teste de Formatos de Imagem

Diferentes formatos de imagem

Formato JPG

Bom para fotografias e imagens com muitas cores e gradientes

Formato PNG

Suporta transparência, bom para gráficos, logos e imagens com texto

Formato GIF

Suporta animações simples, limitado a 256 cores

Formato SVG

Gráficos vetoriais, escaláveis sem perda de qualidade, ideal para ícones e logos

Formato WebP

Formato moderno com melhor compressão, suporta transparência e animação

Comparação de formatos de imagem

Formato Vantagens Desvantagens Melhor uso
JPG/JPEG Boa compressão, amplamente suportado Compressão com perda, sem transparência Fotografias, imagens complexas
PNG Transparência, compressão sem perda Arquivos maiores que JPG Logos, gráficos, imagens com transparência
GIF Animações, amplamente suportado Limitado a 256 cores Animações simples, ícones simples
SVG Escalável, arquivos pequenos Não ideal para fotografias Ícones, logos, gráficos
WebP Melhor compressão, suporta transparência Suporte limitado em navegadores antigos Substituição moderna para JPG e PNG

Sobre este teste

Esta página testa como o crawler lida com imagens em diferentes formatos (JPG, PNG, GIF, SVG, WebP).

O que o crawler deve fazer:

  • Detectar e processar todos os formatos de imagem
  • Identificar o formato de cada imagem
  • Possivelmente sugerir formatos mais eficientes (ex: WebP em vez de JPG)
  • Verificar se há suporte para formatos modernos como WebP
