Dicas de otimização de imagens

  • Dimensionamento adequado: As imagens devem ser servidas no tamanho em que serão exibidas, evitando redimensionamento via HTML/CSS
  • Compressão eficiente: Use ferramentas de compressão para reduzir o tamanho do arquivo sem perda significativa de qualidade
  • Formatos modernos: Considere usar WebP em vez de JPEG/PNG para melhor compressão
  • Lazy loading: Carregue imagens apenas quando estiverem próximas de entrar na viewport
  • Imagens responsivas: Use srcset e sizes para fornecer imagens apropriadas para cada dispositivo

Sobre este teste

Esta página testa como o crawler analisa imagens de diferentes dimensões e níveis de compressão, e se ele consegue identificar oportunidades de otimização.

O que o crawler deve fazer:

  • Detectar as dimensões reais das imagens
  • Comparar as dimensões reais com as dimensões de exibição
  • Identificar imagens que são muito grandes para seu contexto de exibição
  • Analisar o tamanho do arquivo em relação às dimensões
  • Sugerir oportunidades de compressão para imagens grandes
  • Possivelmente recomendar formatos de imagem alternativos
