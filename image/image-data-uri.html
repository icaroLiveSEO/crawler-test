O que é Data URI?

Data URI (Uniform Resource Identifier) é uma forma de incorporar dados diretamente no HTML, em vez de referenciar um arquivo externo. Para imagens, isso significa que o conteúdo binário da imagem é codificado (geralmente em Base64) e incluído diretamente no atributo src.

Formato básico:

data:[<mediatype>][;base64],<dados>

Vantagens

  • Reduz o número de requisições HTTP
  • Útil para pequenos ícones e imagens
  • Não requer hospedagem de arquivos separados
  • Funciona offline

Desvantagens

  • Aumenta o tamanho do HTML
  • Não é cacheado separadamente pelo navegador
  • Codificação Base64 aumenta o tamanho em ~33%
  • Difícil de manter e atualizar

Sobre este teste

Esta página testa como o crawler lida com imagens que usam Data URI em vez de URLs convencionais.

O que o crawler deve fazer:

  • Detectar imagens incorporadas como Data URI
  • Possivelmente extrair e analisar o conteúdo da imagem
  • Identificar o tipo de imagem (SVG, PNG, etc.) a partir do mediatype
  • Possivelmente sugerir quando o uso de Data URI é apropriado ou não
