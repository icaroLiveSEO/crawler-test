Sobre este teste

Esta página testa como o crawler lida com imagens que apresentam diferentes tipos de erros de carregamento.

O que o crawler deve fazer:

  • Detectar imagens que não podem ser carregadas
  • Identificar o tipo de erro (404, DNS, permissão, etc.)
  • Reportar imagens quebradas como problemas de SEO e experiência do usuário
  • Verificar se há texto alternativo adequado para imagens quebradas

Tipos comuns de erros de imagem:

Código de erro Descrição Causa comum
404 Não encontrado Arquivo não existe no servidor
403 Proibido Sem permissão para acessar o arquivo
500 Erro interno do servidor Problema no servidor ao processar a requisição
DNS Domínio não encontrado O domínio não existe ou não está acessível
Timeout Tempo esgotado Servidor muito lento ou não respondeu
