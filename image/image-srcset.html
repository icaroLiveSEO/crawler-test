Como funciona o srcset

O atributo srcset permite especificar múltiplas versões de uma imagem para diferentes condições:

  • Descritores de largura (w): Indicam a largura da imagem em pixels
  • Descritores de densidade (x): Indicam a densidade de pixels (1x = densidade normal, 2x = tela retina)
  • O navegador escolhe automaticamente a melhor imagem com base no tamanho da tela e na densidade de pixels

O atributo sizes ajuda o navegador a determinar qual tamanho a imagem ocupará na tela antes de carregar.

Sobre este teste

Esta página testa como o crawler lida com imagens que usam o atributo srcset para fornecer diferentes versões da mesma imagem.

O que o crawler deve fazer:

  • Detectar todas as variantes de imagem especificadas no srcset
  • Entender a relação entre as diferentes versões da mesma imagem
  • Possivelmente verificar se há versões apropriadas para diferentes dispositivos
  • Identificar o uso de imagens responsivas como uma boa prática
