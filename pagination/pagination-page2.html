Sobre a Paginação

Esta é a segunda página da sequência paginada. Ela contém links para a página anterior e para a próxima página.

Implementação Correta

Esta página implementa links de paginação usando os atributos rel="next" e rel="prev" no cabeçalho HTML.

<link rel="prev" href="pagination-page1.html" /> <link rel="next" href="pagination-page3.html" />

Sobre este teste

Esta página testa como o crawler lida com conteúdo paginado e se ele reconhece corretamente os links de paginação.

O que o crawler deve fazer:

  • Detectar que esta é uma página intermediária de uma série paginada
  • Identificar os links para a página anterior e próxima através dos atributos rel="prev" e rel="next"
  • Seguir os links de paginação para indexar todo o conteúdo
  • Entender a relação entre as páginas da sequência
