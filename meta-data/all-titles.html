Título H1 Completamente Diferente

Esta página possui todos os tipos de título diferentes entre si para testar como o crawler lida com múltiplas opções de título.

Títulos presentes nesta página:

Tipo de Título Valor
title tag Título Principal da Página
meta title Meta Title Diferente
og:title Open Graph Title Único
twitter:title Twitter Card Title Especial
h1 Título H1 Completamente Diferente

Priorização de Títulos

O crawler deve ser capaz de:

  • Identificar todos os diferentes tipos de título
  • Registrar corretamente cada variante
  • Implementar uma lógica de priorização se necessário

Casos de uso para teste:

  1. Verificar se o crawler extrai todos os tipos de título
  2. Verificar qual título é priorizado na exibição de resultados
  3. Testar a capacidade de diferenciar entre os vários tipos de metadados

Alguns crawlers podem priorizar os títulos na seguinte ordem:

  1. Tag title (HTML)
  2. Meta title
  3. Open Graph title
  4. Twitter Card title
  5. Primeiro H1 da página
