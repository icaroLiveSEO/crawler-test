Teste de Links Quebrados

Atenção: Esta página contém links propositalmente quebrados para testar como o crawler detecta e reporta links que levam a recursos inexistentes (404).

Links Quebrados Internos

Links que apontam para páginas inexistentes dentro do mesmo site:

  • Link para página inexistente - deve retornar 404
  • Link para pasta inexistente - deve retornar 404
  • Link absoluto interno inexistente - deve retornar 404

Links Quebrados Externos

Links que apontam para recursos inexistentes em outros sites:

  • Link externo inexistente - deve retornar 404
  • Link para subdomínio inexistente - pode retornar DNS error
  • Link para domínio inexistente - deve retornar DNS error

Links com Erros de Redirecionamento

Links que podem causar problemas de redirecionamento:

  • Link com loop de redirecionamento - pode causar erro de redirecionamento infinito
  • Link com cadeia longa de redirecionamentos - pode causar problemas de performance

Sobre este teste

Esta página contém links quebrados propositalmente para testar como o crawler detecta e reporta problemas de links. Em um site real, links quebrados devem ser evitados, pois:

  • Prejudicam a experiência do usuário
  • Desperdiçam o orçamento de crawl dos motores de busca
  • Podem afetar negativamente o SEO do site
  • Dificultam a navegação e descoberta de conteúdo
