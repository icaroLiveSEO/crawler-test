Sobre este teste

Esta página contém links para recursos não-HTML e links com protocolos não-HTTP. O objetivo é testar como o crawler lida com esses tipos especiais de links.

O que o crawler deve fazer:

  • Detectar links para arquivos não-HTML (PDF, DOCX, etc.)
  • Identificar links com protocolos especiais (mailto:, tel:, etc.)
  • Possivelmente seguir links para arquivos não-HTML para indexação
  • Tratar corretamente links com protocolos não-HTTP
