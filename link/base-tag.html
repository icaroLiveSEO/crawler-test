Teste de Base Tag

Atenção: Esta página contém uma tag <base> que altera a resolução de URLs relativas. A base está definida como https://exemplo.com.br/subdiretorio/.

Tag Base

Esta página inclui a seguinte tag no cabeçalho:

<base href="https://exemplo.com.br/subdiretorio/" />

A tag base define a URL base para todas as URLs relativas na página. Isso afeta links, imagens, scripts, folhas de estilo e outros recursos que usam caminhos relativos.

Links Relativos com Base Tag

Os seguintes links usam caminhos relativos que serão resolvidos em relação à URL base:

Link no HTML URL Resolvida
pagina.html https://exemplo.com.br/subdiretorio/pagina.html
pasta/arquivo.html https://exemplo.com.br/subdiretorio/pasta/arquivo.html
../nivel-acima.html https://exemplo.com.br/nivel-acima.html

Links Absolutos com Base Tag

Os seguintes links usam caminhos absolutos que não são afetados pela tag base:

Link no HTML URL Resolvida
https://www.google.com https://www.google.com
/pagina-raiz.html https://exemplo.com.br/pagina-raiz.html

Sobre este teste

Esta página testa como o crawler lida com a tag base e a resolução de URLs relativas. A tag base é importante para SEO porque:

  • Afeta como os motores de busca resolvem links relativos
  • Pode causar problemas se não for implementada corretamente
  • É útil para sites que são servidos de diferentes caminhos ou domínios

O que o crawler deve fazer:

  • Detectar a presença da tag base
  • Resolver corretamente URLs relativas usando a base especificada
  • Tratar corretamente URLs absolutas independentemente da base

Links de teste:

  • Voltar para a página inicial (URL absoluta)
  • Ir para página de tipos de links (URL relativa)
