Teste de Links com Target Blank

Atenção: Esta página contém links com o atributo target="_blank", que faz com que os links sejam abertos em uma nova aba ou janela do navegador.

Links com Target Blank

Os seguintes links abrem em uma nova aba ou janela:

  • Link interno com target="_blank" - abre a página inicial em nova aba
  • Link externo com target="_blank" - abre o Google em nova aba
  • Link com target="_blank" e rel="noopener" - prática de segurança recomendada
  • Link com target="_blank" e rel="noopener noreferrer" - máxima segurança

Dica de Segurança

Ao usar target="_blank", é recomendado adicionar rel="noopener" ou rel="noopener noreferrer" para evitar vulnerabilidades de segurança. Isso impede que a página aberta tenha acesso à página original através do objeto window.opener.

Sobre este teste

Esta página testa como o crawler lida com links que possuem o atributo target="_blank". Este atributo é comum em sites que desejam manter o usuário na página original enquanto abrem novos conteúdos.

O que o crawler deve fazer:

  • Detectar links com o atributo target="_blank"
  • Seguir esses links normalmente, independente do atributo target
  • Possivelmente identificar e reportar links sem os atributos de segurança rel="noopener" ou rel="noopener noreferrer"

Considerações de SEO:

  • O atributo target="_blank" não afeta diretamente o SEO
  • Os motores de busca seguem os links independentemente de como eles são abertos no navegador
  • O uso de rel="noopener noreferrer" não impede que os motores de busca sigam os links
