Teste de Links via JavaScript

Atenção: Esta página contém links que são injetados dinamicamente via JavaScript. Crawlers que não executam JavaScript não conseguirão detectar esses links.

Links Estáticos (HTML)

Estes links estão presentes no HTML inicial e devem ser detectados por qualquer crawler:

  • Link para página inicial
  • Link para página de tipos de links
  • Link externo estático

Links Dinâmicos (JavaScript)

Estes links serão injetados via JavaScript após o carregamento da página:

Se você está vendo esta mensagem sem links abaixo, o JavaScript não foi executado ou está desabilitado.

Código JavaScript

O seguinte código JavaScript é usado para injetar os links dinâmicos:

// Lista de links a serem injetados const dynamicLinks = [ { url: 'https://www.google.com', text: 'Link dinâmico para Google' }, { url: 'https://www.wikipedia.org', text: 'Link dinâmico para Wikipedia' }, { url: 'broken-links.html', text: 'Link dinâmico interno' }, { url: 'https://exemplo.com.br/pagina-js', text: 'Outro link dinâmico' } ]; // Função para injetar os links function injectLinks() { const container = document.getElementById('js-links-container'); // Limpa o conteúdo existente container.innerHTML = ''; // Cria uma lista não ordenada const ul = document.createElement('ul'); // Adiciona cada link à lista dynamicLinks.forEach(link => { const li = document.createElement('li'); const a = document.createElement('a'); a.href = link.url; a.textContent = link.text; li.appendChild(a); ul.appendChild(li); }); // Adiciona a lista ao container container.appendChild(ul); } // Executa a função quando a página carregar window.onload = injectLinks;

Sobre este teste

Esta página demonstra a diferença entre links presentes no HTML inicial e links injetados via JavaScript. O objetivo é testar como o crawler lida com links que só estão disponíveis após a execução de JavaScript.

O que o crawler deve fazer:

  • Detectar os links estáticos presentes no HTML inicial
  • Se executar JavaScript: detectar também os links injetados dinamicamente
  • Se não executar JavaScript: reportar apenas os links estáticos

Links de teste:

  • Voltar para a página inicial
  • Ir para página de tipos de links
