Sobre esta página

Esta página é acessada através de um link com o atributo title. Ela foi criada para testar como os crawlers processam e utilizam informações adicionais em links.

A URL relativa desta página a partir da raiz do site é: /link/subdir/page4.html

Atributo Title em Links

O atributo title em links HTML fornece informações adicionais sobre o link quando o usuário passa o mouse sobre ele. Embora seja principalmente um recurso de usabilidade, os crawlers também podem extrair essa informação.

Exemplo de link com atributo title:

<a href="page4.html" title="Descrição da página 4">Link com Atributo Title</a>

Benefícios do atributo title em links:

  • Melhora a acessibilidade
  • Fornece contexto adicional para usuários
  • Pode fornecer informações adicionais para crawlers
  • Ajuda usuários a decidir se devem clicar no link

Comparação de Atributos em Links

Atributo Propósito Impacto em SEO
href Define o destino do link Essencial - Define para onde o crawler deve ir
rel="nofollow" Instrui crawlers a não seguir o link Alto - Impede transferência de autoridade
title Fornece descrição adicional Baixo - Pode fornecer contexto adicional
target="_blank" Abre link em nova aba Nenhum - Apenas afeta comportamento do navegador
aria-label Melhora acessibilidade Baixo - Pode ajudar em sinais de qualidade
