Subdiretório - Página 3 (Nofollow)

Sobre esta página

Esta página é acessada através de um link com o atributo rel="nofollow". Ela foi criada para testar como os crawlers lidam com links que têm instruções para não serem seguidos.

A URL relativa desta página a partir da raiz do site é: /link/subdir/page3.html

Atributo Nofollow

O atributo rel="nofollow" é uma instrução para os motores de busca não seguirem o link e não transferirem autoridade ou "link juice" para a página de destino.

Exemplo de link com nofollow:

<a href="page3.html" rel="nofollow">Link com Nofollow</a>

Usos comuns do atributo nofollow:

  • Links em conteúdo gerado por usuários (comentários, fóruns)
  • Links pagos ou patrocinados
  • Links para páginas não confiáveis
  • Links para páginas de login ou registro
  • Widgets e ferramentas de terceiros

Teste para Crawlers

Se um crawler estiver respeitando corretamente o atributo nofollow, ele não deveria descobrir esta página através do link na página de links válidos. No entanto, ele pode descobrir esta página através de outros links no site que não têm o atributo nofollow.

Um crawler avançado deve ser capaz de:

  • Identificar que esta página foi encontrada através de um link com nofollow
  • Registrar o link, mas não seguir automaticamente
  • Não transferir autoridade através desse link
