Subdiretório - Página 1

Sobre esta página

Esta é uma página localizada em um subdiretório da pasta "link". Ela foi criada para testar como os crawlers seguem links para níveis abaixo na estrutura do site.

A URL relativa desta página a partir da raiz do site é: /link/subdir/page1.html

A URL relativa a partir da página de links válidos é: subdir/page1.html

Teste de Profundidade de Crawling

Esta página ajuda a testar a profundidade com que um crawler analisa um site. Alguns crawlers podem ter limites de profundidade configurados, e esta página está um nível mais profundo que a página de links válidos.
