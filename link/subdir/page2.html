Subdiretório - Página 2

Sobre esta página

Esta é a segunda página localizada em um subdiretório da pasta "link". Ela foi criada para testar como os crawlers seguem múltiplos links para o mesmo nível de profundidade.

A URL relativa desta página a partir da raiz do site é: /link/subdir/page2.html

A URL relativa a partir da página de links válidos é: subdir/page2.html

Breadcrumbs

Esta página também demonstra o uso de breadcrumbs (navegação estrutural), que ajudam tanto usuários quanto crawlers a entender a hierarquia do site.

Os breadcrumbs são importantes para:

  • Melhorar a experiência do usuário
  • Ajudar os crawlers a entender a estrutura do site
  • Fornecer links adicionais para páginas importantes
  • Reduzir a taxa de rejeição
