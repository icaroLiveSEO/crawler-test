Teste de Tipos de Links

Links Internos

Links que apontam para páginas dentro do mesmo site:

  • Link interno relativo - usando caminho relativo
  • Link interno com barra - usando caminho absoluto a partir da raiz
  • Link interno absoluto - usando URL completa

Links Externos

Links que apontam para outros sites:

  • Link externo simples - sem atributos especiais
  • Link externo com target="_blank" - abre em nova aba
  • Link externo com rel="nofollow" - indica aos motores de busca para não seguir

Links com Diferentes Atributos Rel

Links com diferentes valores para o atributo rel:

Tipo de Link Descrição
Link com rel="sponsored" Indica links pagos ou patrocinados
Link com rel="ugc" Indica links em conteúdo gerado por usuários
Link com rel="nofollow" Indica que o motor de busca não deve seguir o link
Link com múltiplos rel Combina múltiplos valores de rel

Links com Diferentes Textos Âncora

Links com diferentes tipos de texto âncora:

  • Palavra-chave relevante - texto âncora com palavra-chave
  • Clique aqui - texto âncora genérico (não recomendado para SEO)
  • https://www.example.com/url - URL como texto âncora
  • - imagem como âncora
