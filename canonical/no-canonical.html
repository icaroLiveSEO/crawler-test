Esta página não possui tag canônica

Nenhuma tag link rel="canonical" foi especificada no cabeçalho desta página.

Informação: Quando nenhuma tag canônica é especificada, os motores de busca geralmente consideram a URL atual como a versão canônica do conteúdo.

Sobre este teste

Esta página não contém nenhuma tag canônica. O objetivo é testar como o crawler lida com a ausência de indicação explícita de URL canônica.

O que o crawler deve fazer:

  • Detectar a ausência de tag canônica
  • Considerar a URL atual como a versão canônica (comportamento padrão)
  • Possivelmente reportar como uma oportunidade de melhoria de SEO

Quando a tag canônica é importante:

A ausência de tag canônica pode ser problemática em casos como:

  • Sites com múltiplas URLs para o mesmo conteúdo
  • Sites com parâmetros de URL (ex: ?page=1, ?sort=price)
  • Sites com versões HTTP e HTTPS
  • Sites com versões www e não-www
