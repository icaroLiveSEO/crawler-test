Teste de URL Canônica - Referência Externa

Tag canônica presente nesta página:

<link rel="canonical" href="https://exemplo.com.br/canonical-self.html" />
Atenção: Esta página contém uma tag canônica que aponta para outra URL. Em um ambiente de produção, os motores de busca considerariam a outra URL como a versão principal deste conteúdo.

Sobre este teste

Esta página contém uma tag canônica que aponta para outra página. Isso indica que esta página é uma versão duplicada ou alternativa do conteúdo, e a URL canônica é a versão principal.

O que o crawler deve fazer:

  • Detectar a tag canônica
  • Identificar que a URL canônica aponta para outra página
  • Considerar a outra página como a versão principal do conteúdo
  • Possivelmente não indexar esta página, ou dar preferência à URL canônica nos resultados de busca

Casos de uso para canônica externa:

  • Páginas com parâmetros de URL (ex: filtros, ordenação, tracking)
  • Versões impressas de páginas
  • Versões móveis de páginas (quando não usa design responsivo)
  • Conteúdo duplicado em múltiplos domínios
