Teste de URL Canônica - Relativa

Tag canônica presente nesta página:

<link rel="canonical" href="canonical-self.html" />
Informação: Esta página usa uma URL relativa na tag canônica. Os motores de busca devem resolver esta URL relativa para uma URL absoluta com base na URL atual.

Sobre este teste

Esta página contém uma tag canônica com uma URL relativa. Os motores de busca devem resolver esta URL relativa para uma URL absoluta.

O que o crawler deve fazer:

  • Detectar a tag canônica
  • Resolver a URL relativa para uma URL absoluta
  • Considerar a URL resolvida como a versão canônica

Resolução de URLs relativas:

Se esta página estiver em https://exemplo.com.br/canonical-relative.html, a URL canônica relativa canonical-self.html deve ser resolvida para https://exemplo.com.br/canonical-self.html.

Considerações sobre URLs relativas:

  • URLs relativas são válidas para tags canônicas, mas URLs absolutas são recomendadas
  • URLs relativas podem causar problemas se a página for acessada por diferentes caminhos
  • A maioria dos motores de busca consegue resolver URLs relativas corretamente
