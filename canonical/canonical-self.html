Teste de URL Canônica - Auto-Referência

Tag canônica presente nesta página:

<link rel="canonical" href="https://exemplo.com.br/canonical-self.html" />

Sobre este teste

Esta página contém uma tag canônica que aponta para si mesma (auto-referência). Esta é a implementação correta para páginas que são a versão principal/canônica do conteúdo.

O que o crawler deve fazer:

  • Detectar a tag canônica
  • Identificar que a URL canônica é a própria página
  • Considerar esta página como a versão principal do conteúdo

Propósito da tag canônica:

A tag canônica é usada para:

  • Indicar a versão preferida de uma página quando existem múltiplas URLs com conteúdo similar
  • Evitar problemas de conteúdo duplicado
  • Consolidar sinais de SEO em uma única URL

Quando a tag canônica aponta para a própria página, isso indica que esta é a versão principal do conteúdo.
