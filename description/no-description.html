Página sem Descrições

Atenção: Esta página não possui nenhum tipo de descrição

Esta página foi criada propositalmente sem nenhuma meta description, og:description ou twitter:description.

Sobre este teste

Esta página não contém nenhum tipo de descrição meta. O objetivo é testar como o crawler lida com a ausência de descrições.

O que o crawler deve encontrar:

  • Ausência de meta description padrão
  • Ausência de og:description
  • Ausência de twitter:description

Quando uma página não possui descrição, os motores de busca geralmente extraem um trecho do conteúdo da página para exibir nos resultados de busca.

Um bom crawler deve:

  • Detectar a ausência de descrições
  • Possivelmente gerar uma descrição automática baseada no conteúdo da página
  • Reportar a ausência de descrição como um problema de SEO
