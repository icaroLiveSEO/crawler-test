Teste com Todas as Descrições

Descrições presentes nesta página:

Tipo de Descrição Valor
meta description Esta é a meta description padrão da página.
og:description Esta é a Open Graph description, diferente da meta description padrão.
twitter:description Esta é a Twitter Card description, diferente das outras descrições.

Sobre este teste

Esta página contém todos os três tipos de descrição, cada uma com um conteúdo diferente.

O que o crawler deve encontrar:

  • Presença de meta description padrão
  • Presença de og:description (com conteúdo diferente)
  • Presença de twitter:description (com conteúdo diferente)

Este teste verifica se o crawler consegue extrair e diferenciar os três tipos de descrição.

Priorização de descrições:

Alguns crawlers podem priorizar as descrições na seguinte ordem:

  1. Meta description padrão
  2. Open Graph description
  3. Twitter Card description
