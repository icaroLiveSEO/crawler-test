Teste de Open Graph Description

Descrição presente nesta página:

og:description: Esta página contém apenas uma Open Graph description para testar a extração de descrições pelo crawler.

Sobre este teste

Esta página contém apenas a descrição para Open Graph (property="og:description"). Não possui meta description padrão ou descrição para Twitter Cards.

O que o crawler deve encontrar:

  • Ausência de meta description padrão
  • Presença de og:description
  • Ausência de twitter:description

A Open Graph description é importante para compartilhamento em redes sociais como Facebook.
